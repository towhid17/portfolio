Research Experiences

ENFIRED: ENforcing IoT FIREwall using fog-cloud framework under Delay constraints
Distributed system, Edge-cloud computing, Security
Supervisor: Dr. Rezwana Reaz
My undergrad thesis was on the distribution of IoT firewalls on a fog-cloud-based architecture, titled “ENFIRED: ENforcing IoT FIREwall using fog-cloud framework under Delay constraints”, in which I developed a novel reallocation algorithm for deploying IoT firewalls on a fog-cloud-based architecture to reduce the response time of delay-sensitive IoT devices in a healthcare facility for maintaining the delay under a tolerable limit. I conducted the experiments using the Floodlight SDN controller, mininet, iperf, and the mqtt protocol.

Vein-Based Imaging Approach usign Deep Learning for Real-Time Non-Invasive Dehydration Detection
Machine Learning, Image processing, Ubuquitous computing
Supervisor: Prof. Tanzima Hashem
We implemented a novel dehydration detection system using deep learning models on hand vein images captured by smartphone IR cameras. The system selects the sharpest frame from hand vein videos, focusing on the region of interest (ROI). We developed two ROI extraction methods: a novel geometry-based algorithm and a deep-learning segmentation model. After preprocessing the ROI to highlight veins and reduce noise, we built a dehydration detection model using a multi-modal Siamese network. Now we are finalizing the paper to submit in IMWUT.

Sentence level Bangla Sign Language Dataset creation and Bangla Sign Language Production
Computer Vision, NLP, Machine Learning
Supervisor: Prof. Muhammad Abdullah Adnan
For the Bangla Sign Language Production project, we developed a novel sentence-level dataset tailored for the task. We implemented a GAN model designed to convert skeleton key points into detailed human body representations, where we specifically focused on hand positions and adjusted the loss function accordingly. Currently, we are focused on building a progressive transformer model aimed at enhancing the accuracy and fluency of Bangla sign language production. This transformer architecture is designed to efficiently translate the skeleton key points into coherent and contextually appropriate sign language sequences.
